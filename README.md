### Instruções (PT/BR)
1. Rode dentro da pasta do projeto `bundle`
2. Rode a migrate `rails db:migrate`
3. Carregue os dados do seed `rails db:seed`
4. ligue a api na porta 3001 `rails s -p 3001`
